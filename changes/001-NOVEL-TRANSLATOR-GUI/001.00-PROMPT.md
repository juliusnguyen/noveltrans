# 001 — Novel Translator GUI

## Nguyên văn yêu cầu

Tôi muốn tạo một GUI có thể chạy trên nền Python,

- **Tab 1**: mục tiêu là nhập `input` là URL trang web tiểu thuyết (e.g: https://fanqienovel.com/page/7328655521718733886) sau đó có nút để quét mô tả, số lượng các chương của tiểu thuyết này. Sau đó tải từng chương về.
- **Tab 2**: tuy nhiên do tiểu thuyết là tiếng Trung, vậy nên sẽ có phần chuyển từ tiếng Trung sang Tiếng Việt (hoặc Anh).
- **Tab 3**: export thành các loại file như docx, markdown hoặc epub.

> **Cập nhật 2026-07-04:** người dùng đổi trang web mục tiêu từ fanqienovel.com sang
> `https://www.xbanxia.cc/books/331303.html` (xbanxia.cc — HTML thuần, không mã hoá nội dung).

## Diễn giải mục tiêu

Một ứng dụng desktop GUI viết bằng Python gồm 3 tab:

1. **Scrape / Download** — nhập URL tiểu thuyết (ví dụ fanqienovel.com), quét metadata (tiêu đề, mô tả, tác giả, số chương), rồi tải toàn bộ chương về lưu local.
2. **Translate** — dịch nội dung chương từ tiếng Trung sang tiếng Việt hoặc tiếng Anh.
3. **Export** — xuất kết quả ra các định dạng: DOCX, Markdown, EPUB.
