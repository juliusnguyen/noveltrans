# 009 — Audio from original text — History

## [PLAN] 009.02-INITIAL-PLAN — 2026-07-15

Generated plan: `009.02-INITIAL-PLAN.md`

Add a "Bản dịch / Bản gốc" source toggle to the Audio tab so TTS can voice
`chapter.content` (original), not just `chapter.translated`. Mainly for the Vietnamese
sources (medoctruyen, giatocvuongtai) whose originals are already Vietnamese.

Threads `use_translation: bool` through `AudioWorker` (text/title choice), `pending_audio`
(which column gates pending), and makes the table status/messages source-aware.

Decisions:
- **Track `audio_source` in the DB** (new column, `DEFAULT 'translated'` back-fills
  existing audio) so switching sources re-pends/regenerates correctly — the trivial
  `_migrate()` pattern makes this cheap and avoids a confusing "Đã đủ" on switch.
- **Non-vi originals: allow but warn** (VieNeu is Vietnamese TTS).
- **Default = Bản dịch** (no behaviour change for existing users).

## 2026-07-15 — Implementation

Built the plan.
