# 015 — 69shuba Source Support

Add support for https://www.69shuba.com/book/59024/ as a novel source.

- Recognize `69shuba.com` book URLs and route them to a new source adapter.
- Extract the book metadata (title, author, cover, description) and the chapter list from the book page.
- Extract chapter text from individual chapter pages, cleaned of site chrome/ads.
- Follow the existing source adapter conventions in this repo (registration, selectors, encoding handling, pagination if the chapter list is paginated).
