# 022 — Video quality/speed preset — History

## [FEATURE] One "Chất lượng" setting bundling 3 speed levers — 2026-07-18
After measuring the 1080p render at **~3× realtime** (~1h render / 3h audio), the user asked
to fold the speed levers (720p, lower fps, no spinning vinyl) into a single setting.

**Measured real speeds on this Mac (short 120s slices, production filtergraph):**
- 1080p, 25fps, spinning: **3.04×**
- 720p, 25fps, spinning: **10.6×**
- 720p, 15fps, static disc: **17.1×**

So the presets' `speed` (used only for the confirm-dialog estimate) are set slightly below
measured — 3 / 9 / 15 — so the estimate under-promises under thermal load.

- **`video.py`** — `VIDEO_QUALITY_PRESETS` (high/fast/fastest → width/height/fps/spin_vinyl/
  speed) + `video_preset(key)` (falls back to high). `_filtergraph` gained `spin_vinyl`:
  when False it overlays the disc **statically** (`[0:v][2:v]overlay=…`) instead of the
  per-frame `rotate` — the single biggest filter cost — so the disc is still shown, just
  not spinning. `render_video` gained `spin_vinyl`, threaded to the filtergraph.
- **`config.py`** — `video_quality` property (persisted), `DEFAULT_VIDEO_QUALITY = "high"`.
- **`workers.py`** — `VideoWorker` gained `spin_vinyl`, passed to `render_video`.
- **`tab_audio.py`** — a "Chất lượng" dropdown (persists on change); `_start_video` resolves
  the preset → width/height/fps/spin and shows an **estimated render time** in the confirm
  dialog (audio hours ÷ preset speed).
- Tests: presets cover the three tiers with ascending speed + unknown-key fallback;
  `_filtergraph(spin_vinyl=False)` has no `rotate` and uses the static overlay. 504 passing
  (from 501).

### Verified with real renders
Rendered 120s at each preset: **fast 10.6×**, **fastest 17.1×** realtime (both valid ffmpeg,
static-vinyl filtergraph works). A 720p "fastest" frame confirmed the layout scales cleanly
(PlayerLayout is proportional) — photo left, static fox-label disc, titles, bars, knob, no
overlap. Estimates now shown: e.g. the 19.6h novel ≈ 6.5h (high) / ~2.2h (fast) / ~1.3h
(fastest).
