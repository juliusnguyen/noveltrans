> User request (2026-07-05):

Sử dụng repo này để thêm một tính năng chuyển toàn bộ văn bản được dịch thành audio:
`https://github.com/pnnbao97/VieNeu-TTS`

## Intent

Convert a novel's **translated text** (Vietnamese) into audio using **VieNeu-TTS**
(a Vietnamese text-to-speech model), so the user can listen to translated chapters
as an audiobook.

## Open questions (resolve in /plan)

- How VieNeu-TTS is invoked: Python API, CLI, HTTP server? Hardware needs on the
  user's M4 24 GB Mac? (research the repo first)
- Integration shape: a new Exporter ("Audio / MP3-WAV") in the existing exporter
  registry vs a dedicated Tab. The exporter ABC currently returns one file —
  audiobooks likely need one file per chapter (a folder) or m4b concatenation.
- Per-chapter resumability like download/translate (chapters are long; TTS is slow).
- Voice selection / speed; output format (wav/mp3/m4b); where files land
  (project `exports/` dir).
