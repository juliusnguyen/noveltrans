# 003 — Support for medoctruyen.vn

Add support for the source site `https://medoctruyen.vn/`.

Example novel URL to develop and test against:
`https://medoctruyen.vn/tu-bao-tien-bon`

## Scope

- New scraper alongside the existing `ixdzs` and `xbanxia` scrapers in
  `src/noveltrans/scrapers/`, registered so the GUI's Scrape tab accepts
  `medoctruyen.vn` novel URLs.
- Must produce the same `Novel` / `Chapter` shapes the rest of the pipeline
  (translate → export → TTS) already consumes: novel metadata (title, author,
  cover, description) plus the full ordered chapter list and per-chapter text.
- Handle whatever the site actually does for chapter listing and chapter body
  fetching (pagination, JSON/XHR endpoints, or server-rendered HTML) — inspect
  the live page before assuming.

## Open questions to resolve during planning

- medoctruyen.vn appears to serve Vietnamese text, whereas the existing sources
  are Chinese. Decide whether the translate step should be skipped/optional for
  this source, or whether the source language should become an explicit field on
  the novel so downstream steps (translation prompt, TTS voice) behave correctly.
