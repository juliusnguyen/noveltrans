# Implementation Plan: Strip special characters before TTS synthesis

## Summary
Add one pure function `clean_for_tts(text: str) -> str` in a new module `src/noveltrans/tts/clean.py`, a whitelist-based cleaner that preserves Vietnamese (precomposed and combining), ASCII, digits, and prosody punctuation while stripping everything else. Call it at the single chokepoint `TtsEngine.synthesize_chapter` (base.py), gated by a new `clean: bool = True` parameter. Wire a `tts_clean_text` config bool (default True) through `AudioWorker` and a Settings checkbox. Cleaning applies only to the in-memory copy fed to the engine; stored `translated`/`content` are never touched.

Scope is deliberately small: one pure function + one param + a toggle + wiring + one new test file.

## The keep-set (the crux), specified precisely

Use stdlib `unicodedata` only. `regex` is **not** a dependency (confirmed in `pyproject.toml`), and adding it just for `\p{Script=Latin}` isn't worth it.

**Step 0 ‚Äî normalize before filtering:**
- Apply `unicodedata.normalize("NFC", text)`. For typical translator output (already NFC) this is a no-op, so it does not break the byte-identity guarantee; it also composes any stray combining marks so the base+mark stays coherent.
- Map dashes to ASCII hyphen: em-dash `‚Äî` (U+2014), en-dash `‚Äì` (U+2013), horizontal bar `‚Äï` (U+2015) ‚Üí `-`.
- Map every Unicode space separator (category `Zs`, e.g. non-breaking space U+00A0) plus `\t \r \f \v` ‚Üí regular space. Preserve `\n` untouched.

**Step 1 ‚Äî per-character keep predicate:**

```
def _keep(ch):
    if ch in _ALLOWED: return True          # prosody punctuation + space + \n
    cat = unicodedata.category(ch)
    if cat[0] == "L": return _is_latin(ch)  # Vietnamese + ASCII kept; CJK/Han dropped
    if cat[0] == "M": return True           # combining marks (decomposed tone marks)
    if cat == "Nd" and ch.isascii(): return True
    return False

def _is_latin(ch):
    try: return unicodedata.name(ch).startswith("LATIN")
    except ValueError: return False
```

Why this preserves Vietnamese and drops Han:
- Every precomposed Vietnamese letter has a Unicode name starting with `LATIN` (e.g. `·ªô` = "LATIN SMALL LETTER O WITH CIRCUMFLEX AND DOT BELOW", `ƒë`/`ƒê` = "LATIN ‚Ä¶ LETTER D WITH STROKE"). All kept.
- Han = "CJK UNIFIED IDEOGRAPH-‚Ä¶" ‚Üí does not start with `LATIN` ‚Üí dropped. The script-aware discrimination a pure "keep all L*" would miss.
- Emoji / `‚òÖ ‚óÜ ‚Äª „Äê „Äë` / arrows = symbol categories ‚Üí dropped. Zero-width / control (`Cf`/`Cc`) ‚Üí dropped. Markdown `* # _ \`` ‚Üí dropped.

**Prosody punctuation allow-set (cross-checked against `_SENTENCE_RE`):**
`base.py` line 13 is `_SENTENCE_RE = re.compile(r"[^.!?‚Ä¶]*[.!?‚Ä¶]+[\"'‚Äù‚Äô)]*\s*|[^.!?‚Ä¶]+$")`. Chunking depends on the enders `. ! ? ‚Ä¶` and the trailing group `" ' ‚Äù ‚Äô )`. The allow-set MUST be a superset:
- Sentence enders: `. ! ? ‚Ä¶`
- Clause/pause: `, ; :`
- Quotes (straight + curly): `" ' ‚Äú ‚Äù ‚Äò ‚Äô`
- Parentheses: `( )`
- Hyphen `-` (dashes normalized into this in Step 0)
- Whitespace: regular space and `\n`

**Step 2 ‚Äî tidy the holes:** collapse runs of spaces/tabs to one space (never touch `\n`); strip trailing spaces per line; preserve `\n\n` (what `split_sentences` splits on); final `.strip()`.

**Edge cases (documented, not blocking):** full-width digits/Latin dropped (assumed absent from VN novels); bare combining mark kept (harmless); `% $ &` stripped not expanded ‚Äî `"100%"` ‚Üí `"100"`, a known limit of the strip-only design.

## Pure function: signature and location
- Module **`src/noveltrans/tts/clean.py`** (beside `base.py`). No Qt/sqlite/engine ‚Äî mirrors `find_replace.py` / `translators/names.py`.
- **`def clean_for_tts(text: str) -> str`**. Deterministic, idempotent, testable without the ~334MB model.

## Plumbing decision: inside `synthesize_chapter` (recommended)

Clean inside `TtsEngine.synthesize_chapter`, gated by `clean: bool = True`, immediately before `split_sentences`:
```
text = f"{title}\n\n{body}" if title else body
if clean:
    text = clean_for_tts(text)          # new
chunks = split_sentences(text, self.max_chunk_chars)
```
Rationale over worker-side cleaning:
- **Single chokepoint** ‚Äî both `AudioWorker` paths (sequential `workers.py:425`, parallel pool `:487`) plus the CLI (`__main__.py:49`) route through here; the CLI benefits for free.
- **Correctness ordering** ‚Äî chunking depends on kept punctuation; cleaning the exact string `split_sentences` consumes, in the same place, keeps them from drifting.
- **Purity preserved** ‚Äî the engine takes a plain `bool`, not `AppConfig`. The flag rides in like `cancelled` does.

Toggle path: `AppConfig.tts_clean_text` ‚Üí `AudioWorker.clean_text` ‚Üí `clean=self.clean_text` at both call sites.

## Testing strategy
New pure suite `tests/test_tts_clean.py`:
- **Vietnamese preservation (most important):** a real VN paragraph (precomposed tone marks, punctuation, `\n\n`) ‚Üí byte-identical out.
- CJK removal; decorative/emoji removal (`"‚òÖ Ch∆∞∆°ng 1 ‚òÖ"` ‚Üí `"Ch∆∞∆°ng 1"`); zero-width/control removal; markdown removal.
- **Prosody kept + splitting works:** feed cleaned output into `split_sentences`, assert sentence boundaries survive and specials are gone.
- Paragraph/`\n\n` preservation; space collapse; dash normalization; idempotence; empty-after-clean (`split_sentences(clean_for_tts("‚òÖ‚òÖ‚òÖ")) == []`, proving `synthesize_chapter`'s existing empty-`TtsError` fires gracefully).

Extend `tests/test_tts.py` with the existing `FakeTtsEngine`: `clean=True` strips before chunking; `clean=False` passes text through unchanged (proves wiring + escape hatch).

## Non-persistence confirmation
Cleaning touches only the local `text` in `synthesize_chapter`; the DB write path (`save_audio`) stores path/voice/duration/source, never text. `chapter.translated`/`content` are read-only inputs. Fully non-destructive.

## Risks / uncertainties flagged for the user
- **NFC normalization** ‚Äî recommended (composes stray marks; no-op for typical NFC input so byte-identity holds). If you'd rather guarantee zero normalization, drop the NFC step and lean on the `M*`-keep branch. **Your call.**
- **`% $ &` stripped, not spoken** ‚Äî accepted per scope; `"100%"` ‚Üí `"100"`. A tiny expansion map is the follow-up if it bites.
- **Full-width Latin/digits dropped** ‚Äî assumed absent from VN novel text.
- Optional `--no-clean` CLI flag is a nice-to-have.
