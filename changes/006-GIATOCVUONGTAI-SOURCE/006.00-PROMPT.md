# 006 — giatocvuongtai.com source

```
https://giatocvuongtai.com/stories/dm-edit-chao-mung-den-voi-phong-livestream-ac-mong/
```

## Scope (to confirm during /plan)

- New `SiteAdapter` for `giatocvuongtai.com` (metadata + table of contents + chapter
  body), registered so pasting a `giatocvuongtai.com` story URL routes to it.
- Match the existing adapter contract (see `src/noveltrans/scrapers/` — base + the
  medoctruyen adapter as the reference implementation).
- Handle the site's realities: auth/cookies if full chapter bodies require login,
  pagination of the TOC, and any per-day / rate limits (mirror the medoctruyen
  DailyLimitError / RateLimitedError handling only if the site actually imposes them).
- Tests with saved HTML fixtures for TOC + chapter parsing.
