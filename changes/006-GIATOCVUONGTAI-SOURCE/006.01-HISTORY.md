# 006 — giatocvuongtai.com source — History

## [PLAN] 006.02-INITIAL-PLAN — 2026-07-15

Generated plan: `006.02-INITIAL-PLAN.md`

Did live API reconnaissance first (site is an Astro SPA, so HTML scraping is a dead
end): the chapter list and bodies come from a clean **public JSON API** with no auth,
no obfuscation, and no observed rate limit.
- Story + full TOC: `GET /api/public/story/<slug>.json` (all 745 chapters, no pagination).
- Chapter body: `GET /api/public/chapter/<id>.json`, content as `{blocks:[{type,inline:[{text}]}]}`.
- Content is already Vietnamese ("edit") → `source_lang="vi"` like medoctruyen.

Plan scope: one new adapter file + a one-line registration edit + JSON-fixture tests.
No auth/cookies/rate-limit/GUI/config changes needed. Two design decisions recorded:
parse JSON in-adapter (no `base.py` change), and store the chapter-content API URL
directly in `ChapterRef.url`.

## 2026-07-15 — Implementation

Built the plan as written.

**Changes:**
- `src/noveltrans/scrapers/giatocvuongtai.py` (new): `GiatocvuongtaiAdapter` — pure
  JSON, no BeautifulSoup. `_slug` + `_get_json` (unwraps the `{data,error}` envelope);
  `fetch_metadata`/`fetch_chapter_list` off one story endpoint; `fetch_chapter` off the
  per-chapter endpoint; `_blocks_to_text` flattens `blocks→inline→text` with an
  allow-list (`paragraph`/`heading`) so unknown block types are skipped. Handles
  `content` as dict or JSON string. `source_lang="vi"`.
- `src/noveltrans/scrapers/__init__.py`: registered `giatocvuongtai` in the import line.
- `tests/fixtures/giatocvuongtai/`: `story.json` (trimmed to 3 chapters incl. one
  unpublished, captured live), `chapter.json` (first chapter, 5 blocks, live),
  `chapter_stringcontent.json` (content as JSON string), `chapter_mixed_blocks.json`
  (hand-crafted heading/divider/image/empty-para to prove defensiveness).
- `tests/test_scrapers_giatocvuongtai.py`: registry/metadata/chapter-list/content
  (dict, string, mixed, empty, missing)/errors/slug, plus a `@pytest.mark.live` test.

**Verification:** ruff clean; 195 offline tests pass + the live test passes. Live
end-to-end confirmed: sample URL routes to the adapter, 745 chapters, chapter 1 is
175 clean Vietnamese paragraphs, no raw JSON leakage, no auth prompt.
